# roq-coinbase-pro

Installs Roq Coinbase/Pro Gateway.

## Requirements

* Conda
* Docker (optional)

## Variables

* `coinbase_pro_use_docker` (bool)
  * `false` = Conda installation
  * `true` = Docker installation

## Tasks

* Create directories
* Create run scripts
* Install roq-coinbase_pro
* Create systemd services
